##  프로젝트 경험

###  금융 P2P 송금 시스템
**개인 프로젝트 | 6개월 소요** [![GitHub](https://img.shields.io/badge/GitHub-Repository-181717?style=flat&logo=github)](https://github.com/dbwndnjs32212804/personal-finance-dashboard)

**사용 기술:** Java, Spring Boot, MySQL, JPA, @Transactional

**프로젝트 개요:** 금융 IT 시스템의 안정성과 데이터 일관성을 목표로 개발한 P2P 송금 시스템

**주요 성과:**
- **트랜잭션 관리:** `@Transactional`을 적용하여 송금 과정(출금, 입금, 기록)의 원자성 보장
- **보안 문제 해결:** 403 Forbidden 에러 발생 시, 로그 분석을 통해 '인증'과 '인가'의 차이를 파악하고 API 환경에 맞는 보안 규칙 재설정
- **의존성 디버깅:** `@NotBlank` 어노테이션 미작동 문제 발생 시, `pom.xml`의 의존성 누락을 원인으로 파악하고 해결
